贪吃蛇（也叫做贪食蛇）游戏是一款休闲益智类游戏，既简单又耐玩，唯一的目标就是做这条 gai 上最长（pang）的蛇（zhu）！

玩贪吃蛇最考验的就是走位（简称蛇皮走位），各种漂移过人，唉，就是玩儿！
