# 前言

井字棋，英文名叫 Tic-Tac-Toe，是一种在 3*3 格子上进行的连珠游戏，由于棋盘一般不画边框，格线排成井字故得名。（直接百度照抄哈哈哈 ）

另外，百度还给出了一些攻略，反正我是没看懂。。。（想不到小小井字棋都要勾心斗角 ）

当然，在这个小游戏中，要让电脑像人一样智能是不太可能的，因为要涉及到很多的技术。因此，为了简化游戏，电脑下棋的位置将采用随机数的方式来确定。
